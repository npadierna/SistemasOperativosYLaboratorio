Web Application
==============================

Respositorio contendor del proyecto Web final para la asignatura Sistemas Operativo y Laboratorio.
La aplicación Android extraerá y/o modificará los datos necesarias para el funcionamiento desde esta aplicación.

Versiones de las tecnologías utilizadas para la creación del proyecto:
 - GlassFish Server 3.1.2.2
 - Hibernate ORM 4.2.3
 - Java version 1.7.0_51
 - Java(TM) SE Runtime Environmet (Build 1.7.0_51-b00)
 - Jersey - RESTful Web Services 1.13
 - MySQL 5.5.35
 - PrimeFaces 3.5
 - Spring Framework MVC 4.0.1
