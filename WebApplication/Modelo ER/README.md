Entity-Relationship Model
==============================

Respositorio contendor del proyecto del Modelo Entidad/Relación para la asignatura Sistemas Operativo y Laboratorio.
El proyecto es realizado con la herramienta: MySQL Workbench, igualmente contiene una imagen del modelo relacional.

Versiones de las tecnologías utilizadas para la creación del proyecto:
 - MySQL Ver 14.14 Distrib 5.5.35
 - MySQL Workbench 5.2.47 CE Revision 10398
